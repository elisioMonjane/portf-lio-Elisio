Elísio Monjane

Biografia

Elísio Monjane é um talentoso especialista em design de interfaces de usuário (UI) e web design, com uma paixão por criar experiências digitais envolventes e funcionais. Atualmente, ele combina sua experiência prática no uso de ferramentas como Figma e habilidades em design responsivo com sua formação acadêmica em Informática pela Universidade Eduardo Mondlane (UEM), em Moçambique. Com um olhar atento para detalhes, Elísio Monjane é conhecido por transformar ideias complexas em interfaces intuitivas e visualmente atrativas. Ele tem experiência no desenvolvimento de protótipos interativos, design para aplicativos móveis e criação de websites modernos, sempre priorizando a acessibilidade e a experiência do usuário (UX).

Skills


  • Design de Interfaces (UI): Especialista no uso de Figma para criar protótipos interativos e design de alta fidelidade.

  • Web Design: Criação de sites responsivos com HTML, CSS e frameworks como Bootstrap e Tailwind CSS.


  • Desenvolvimento Web: Conhecimento em JavaScript e CMS como WordPress para integração de designs em plataformas funcionais.
